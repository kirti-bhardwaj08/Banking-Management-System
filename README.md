# Banking-Management-System
A robust, console-based Banking Management System developed in C++ that demonstrates core concepts of Object-Oriented Programming (OOP), file handling, and data persistence. This project simulates a real-world banking environment where users can manage their finances through a secure interface.

* Transaction Tracking: Real-time tracking of deposits and withdrawals stored in a std::vector and synced to permanent storage.
